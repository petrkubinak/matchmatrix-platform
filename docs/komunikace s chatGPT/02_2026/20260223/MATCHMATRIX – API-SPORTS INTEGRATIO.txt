🔑 API

Používám API-Sports dashboard (ne RapidAPI)

Base URL:

https://v3.football.api-sports.io

Header:

x-apisports-key: <SECRET>

Limit: 100 requestů / den

Postup:

GET /teams?league=<id>&season=2026

GET /fixtures?league=<id>&season=2026
