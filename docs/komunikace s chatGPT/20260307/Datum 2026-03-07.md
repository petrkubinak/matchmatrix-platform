staging.players_import

birth_date

nationality

3️⃣ Vyřešen problém s JSON quoting

Původní verze skriptu ukládala:

raw_json

→ způsobovalo chyby

unterminated quoted string
extra command-line argument ignored

řešení:

odstraněno raw_json

ukládají se pouze normalizovaná pole

4️⃣ Merge hráčů do canonical tabulky

spuštěn skript:

009_merge_players.sql

pipeline:

API-Football
      ↓
staging.players_import
      ↓
009_merge_players.sql
      ↓
public.players
5️⃣ Vyřešen problém s duplicitami

chyba:

duplicate key value violates unique constraint
ux_players_ext_source_player_id

řešení:

deduplikace ve staging

merge přepsán na logiku:

UPDATE existing
INSERT new

❌ Problémy řešené během dne
1️⃣ quoting v psql

unterminated quoted string

vyřešeno odstraněním JSON pole.

2️⃣ duplicate key při merge
duplicate key value violates unique constraint

vyřešeno:

deduplikací

změnou merge logiky

3️⃣ ON CONFLICT error
there is no unique constraint matching ON CONFLICT

→ schema nemělo odpovídající constraint

řešení:

UPDATE + INSERT pattern
