# MatchMatrix – pracovní zápis

## Datum: 15.03.2026

---

# 1. Cíl dne

Cílem dne bylo **dokončit kompletní ingest pipeline pro hráče (Players pipeline)** a připravit architekturu pro **rozšiřování hráčských dat z více zdrojů**.

Do dneška MatchMatrix uměl:

* ingest lig
* ingest týmů
* ingest zápasů
* základní ingest hráčů

Dnes jsme systém posunuli tak, aby:

* uměl **stahovat hráče z API-Football**
* ukládal **raw payloady**
* prováděl **staging normalizaci**
* prováděl **merge do canonical player registry**
* umožňoval **další enrichment hráčských dat z dalších zdrojů**

Tím jsme připravili základ pro **player intelligence layer**, který bude později využívat Ticket Engine.

---

# 2. Finální architektura Players pipeline

Finální pipeline dnes vypadá takto:

```
API-Football
   /players
        │
        ▼
staging.stg_api_payloads
        │
        ▼
staging.players_import
        │
        ▼
staging.stg_provider_players
        │
        ▼
public.players
        │
        ▼
public.player_provider_map
```

Tato pipeline byla dnes plně zprovozněna.

---

# 3. API-Football Players ingest

Byl vytvořen worker:

```
pull_api_football_players_v4.py
```

Umí:

* stahovat endpoint

```
/players
```

* paging
* ukládání raw payload
* ukládání do `players_import`

Výsledek prvního ingestu:

```
players_import: 540
```

z toho:

```
league=95
season=2024
players=60
```

---

# 4. Players Bridge

Byl vytvořen bridge worker:

```
run_players_bridge_v4.py
```

Pipeline:

```
staging.players_import
→ staging.stg_provider_players
```

Funkce:

* deduplikace hráčů
* normalizace provider dat
* příprava unified staging vrstvy

Výsledek:

```
stg_provider_players: 533
```

---

# 5. Merge do canonical players

Byl vytvořen worker:

```
run_players_public_merge_v2.py
```

Pipeline:

```
stg_provider_players
→ public.players
→ public.player_provider_map
```

Výsledky:

```
public.players: 533
public.player_provider_map: 533
```

Statistika merge:

```
processed rows: 533
inserted players: 58
updated players: 475
```

Tím vznikl **canonical registry hráčů**.
